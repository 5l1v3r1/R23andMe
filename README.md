# R23andMe
23andMe statistical analysis in R

Import the code:

    source('23andme.r')

Histogram by base pair:

    histogram_bybase('genome_Dario_Clavijo_Full_20150805164958.txt')

Histogram by gen:

    histogram_bygen('genome_Dario_Clavijo_Full_20150805164958.txt',c(1:22,'X','Y','MT'))
